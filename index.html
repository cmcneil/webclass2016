Goals

The objective of the class is to provide students with all of the following:

  • An introductory foundation to how the internet works.
    • What is a server?
    • What is a protocol?
  • Familiarity with common technology used in servers, and the basics of their hardware.
    • Each student will manage their own server, a Rasberry Pi.
    • Familiarity with Debian, an Operating System used by companies and individuals for servers worldwide.
    • What is RAM? What is the CPU? What happens when I run a program?
    • What happens when I type www.google.com into my browser?
  • Ability to create simple webpages, and basic web technologies.
    • GitHub
    • HTML
  • Some fun with hardware
    • Controlling lights over a network.

Why These Technologies?

There are many different technologies that one could choose to teach an internet class. Here's why we choose to work with the technologies we do.

Raspberry Pi

The Raspberry Pi is ideally suited to education for many reasons. It is around the price of an Arduino ($30), yet it is actually a full server, running a Linux (Debian) Operating System. Therefore it can be used both to learn hardware, like an Arduino, but it also allows each student to manager their own server, which is extremely valuable to the learning process. In addition, the Pi uses SD cards for persistent memory, making it easy to have some spare SD cards on reserve, and give them to students who break their systems. In this class, we will have SD images of various system states, so we can always easily bring students who fall behind back up to speed. The Raspberry Pi is also used for projects worldwide among DIY (Do It Yourself) and Maker communities.

Debian

Debian OS is one of several Linux distributions that form the worldwide industry standard for server OSs. Debian is free, open source, and ideally suited for lightweight servers. Debian is also the best way to begin learning Linux, the primary OS used in servers, smartphones, routers, watches, etc.

GitHub

GitHub is used as a platform for code storage and collaboration worldwide. It is both useful, and a gateway to the software community.
